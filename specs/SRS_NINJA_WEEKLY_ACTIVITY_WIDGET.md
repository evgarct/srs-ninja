# SRS Ninja --- Feature: Weekly Activity Widget

## Context

На Home Screen нужен компактный индикатор регулярности обучения за 7 дней.

Виджет должен показывать:
- недельную активность по дням,
- текущий streak,
- сколько слов закрепилось после тренировки в каждый активный день.

------------------------------------------------------------------------

## What to Build

### 1. Weekly Activity Widget (Home Screen)

Добавить на главную страницу блок `Activity`, который отображает активность пользователя за последние 7 дней.

### 2. Visual Layout

Виджет состоит из двух частей:
- верхняя строка: `🔥 X day streak!`
- нижний блок: 7 чипов дней недели (Mon..Sun)

Для каждого дня:
- сверху подпись дня недели,
- внутри круглого чипа:
  - число закреплённых слов, если `masteredWords > 0`,
  - `✓`, если день активен, но `masteredWords = 0`,
  - серый эмодзи (например `😶`), если день неактивен.

### 3. Block States

Каждый день имеет состояния:

| State | Meaning |
| --- | --- |
| empty | пользователь не делал reviews |
| active | пользователь сделал >=1 review |
| today | текущий день подсвечен отдельной тёмно-зелёной обводкой |

### 4. Streak Indicator

`streak` --- количество последовательных активных дней, идущих назад от текущего дня.

Пример:
- Wed active
- Thu active
- Fri active
- Sat empty
- Sun active (today)

Streak = `1` (считается только непрерывная последовательность от today).

### 5. Activity Definition

День считается активным, если пользователь сделал хотя бы 1 review.

Источник: таблица `reviews`.

------------------------------------------------------------------------

## Data Query

Минимально необходимые поля:

```sql
SELECT reviewed_at, card_id, rating, state
FROM reviews
WHERE user_id = '{user_id}'
  AND reviewed_at >= '{utc_start_one_day_before_local_week_start}'
ORDER BY reviewed_at ASC;
```

Примечания:
- агрегирование по дням выполняется в приложении с учётом timezone пользователя,
- после выборки записи дополнительно фильтруются по явному набору local week date keys,
- это защищает от потери ранних утренних reviews в timezone восточнее UTC,
- UI показывает ровно последние 7 календарных дней.

------------------------------------------------------------------------

## Mastered Words Logic

Для каждого дня считается `masteredWords`:
- учитываются уникальные `card_id` за день,
- учитываются только записи `rating >= 3` и `state = 'review'`.

`masteredWords` используется как индикатор «слова закрепились после тренировки».

## Behavior

- активные дни отображаются зелёными чипами,
- неактивные дни отображаются нейтральными чипами с серым эмодзи,
- текущий день имеет тёмно-зелёную обводку,
- число закреплённых слов показывается внутри чипа только если `masteredWords > 0`,
- для каждого чипа есть hover-tooltip с объяснением:
  - сколько слов закрепилось,
  - что они в стадии `review`,
  - сколько было повторений в день.

------------------------------------------------------------------------

## Edge Cases

- новый пользователь --- все дни пустые,
- 1 review --- день активный,
- несколько reviews одной карточки за день --- считается 1 закреплённое слово (уникальный `card_id`),
- timezone пользователя должен учитываться,
- если сегодня не было активности, streak = 0.
